# output/

This directory is the build target for the static ADS edition.

It is not tracked by Git. Run `build_server.py` from the `build/` directory
to populate it. The generated files are published via the separate
[achterdeschermen](https://github.com/centre-for-manuscript-genetics/achterdeschermen)
repository and archived on Zenodo.

See the root `README.md` for full build instructions.
